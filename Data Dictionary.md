# Data Dictionary: Global YouTube Statistics

This document describes the columns available in the `Global YouTube Statistics.csv` used for the Social Media Analytics project.

| Column Name | Data Type | Description |
| :--- | :--- | :--- |
| **rank** | Integer | Position of the YouTuber based on subscriber count. |
| **Youtuber** | String | Name of the YouTube channel. |
| **subscribers** | Integer | Total number of subscribers. |
| **video views** | Float | Total views accumulated across all videos on the channel. |
| **category** | String | The niche/genre of content (e.g., Music, Games, Education). |
| **Title** | String | Official title of the channel. |
| **uploads** | Integer | Total number of videos uploaded. |
| **Country** | String | Origin country of the YouTuber. |
| **Abbreviation** | String | Two-letter ISO country code. |
| **channel_type** | String | Classification of the channel's content style. |
| **video_views_rank** | Float | Global rank based on total video views. |
| **country_rank** | Float | Rank of the YouTuber within their specific country. |
| **channel_type_rank**| Float | Rank of the YouTuber within their channel category. |
| **video_views_for_the_last_30_days** | Float | Total views gained in the most recent 30-day period. |
| **lowest_monthly_earnings** | Float | Minimum estimated monthly revenue in USD. |
| **highest_monthly_earnings**| Float | Maximum estimated monthly revenue in USD. |
| **lowest_yearly_earnings** | Float | Minimum estimated annual revenue in USD. |
| **highest_yearly_earnings**| Float | Maximum estimated annual revenue in USD. |
| **subscribers_for_last_30_days** | Float | New subscribers gained in the last 30 days. |
| **created_year / month / date** | Float | The specific date the YouTube channel was established. |
| **Gross tertiary education enrollment (%)** | Float | Percentage of population enrolled in higher education in the YouTuber's country. |
| **Population** | Float | Total population of the YouTuber's country. |
| **Unemployment rate** | Float | National unemployment rate of the host country. |
| **Urban_population** | Float | Number of people living in urban areas in the host country. |
| **Latitude / Longitude**| Float | Geographical coordinates for mapping. |
